1. semester MMD

Windsurfingklubben Seven Winds (WSW)

KRAV:

"Der skal laves en fuld designproces til websiden, inden kodning startes (bemærk: planlæg efter vejledningsdage)
Websiden skal kodes i HTML5 og CSS i PHPStorm eller anden moderne IDE
■ Der må ikke benyttes frameworks som Bootstrap eller Materialize mv.
■ Der må ikke benyttes open source systemer (f.eks. Wordpress) eller drag & drop løsninger (f.eks. WIX)
Websiden skal være responsive og gøre brug af
■ Flexbox og CSS-grid
■ Media queries
■ %-bredder
■ Mobile-first"

HTML, CSS

2. semester MMD

Temaopgave 2 - Dominio de Pontas

KRAV:

"I skal kode websiden ved hjælp af HTML5 og CSS (gerne med brug af SCSS), så den matcher jeres designoplæg og målgruppe mm fra temaopgave 1. Jeres første opgave her er at sikre, at den kodede version ligner jeres prototype, og dermed også jeres arbejde med detaljer i CSS.
Siden skal være responsive og den skal være mobileFirst. Den skal være velfungerende på mobil, tablet og computer, og da koden skal være mobileFirst, skal I starte med at lave CSS til mobil, og så lave breakpoints med media queries så den ser pæn ud på tablet og computer.
I skal benytte JQuery til flere elementer på siden."


jQuery, HTML, CSS
